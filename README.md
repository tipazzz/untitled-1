## Use this extension

This repository can be added as an **extension** in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/shakao/pxt-tutorial-minecraft** and import

## Edit this extension ![Build status badge](https://github.com/shakao/pxt-tutorial-minecraft/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/shakao/pxt-tutorial-minecraft** and click import
